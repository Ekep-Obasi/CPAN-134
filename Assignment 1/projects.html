The Testify Dashboard is a modern web application built to provide an intuitive interface for users to collect, manage, and share testimonials. Utilizing the Next.js framework, this frontend is designed for seamless user experience.
Visit Now!

Features

  • User Authentication: Seamless signup and login processes, including options for password recovery.
  • Dashboard: An interactive dashboard for managing testimonials, projects, and settings.
  • Responsive Design: Optimized for both desktop and mobile devices.
  • Customizable: Users can personalize their experience and manage their profiles easily.
  • Project Management: Create projects to organize and manage testimonials and forms.
  • Testimonial Importing: Import testimonials manually by pasting links to the sources (e.g., Twitter posts).
  • Form Customization: Create and fully customize forms to collect testimonials from clients.
  • Embed Features: Embed forms and testimonials on external sites using a provided embed script.
  • Social Media Content Generation: Generate image posts and video compilations with the testimonials collected.

Tech Stack

  • Framework: Next.js (with TypeScript)
  • Styling: Tailwind CSS
  • State Management: TanStack React Query
  • API Integration: Axios for communicating with the Testify backend
  • Video Generation: Remotion for video compilations
  • Data Fetching: TanStack React Query

Installation

  1. Clone the Repository:

    git clone https://github.com/Ekep-Obasi/app.testify.git
cd app.testify

  2. Install Dependencies:

    pnpm install

  3. Set Up Environment Variables:

    Copy .env.example to .env and add the required configurations. 

    cp .env.example .env

  4. Start the Development Server:

    pnpm run dev

    The application should now be running at http://localhost:3000.

Usage

Once the application is up, users can:

  • Register or log in to their accounts.
  • Access the dashboard to manage testimonials and projects.
  • Create a project under which they can organize and manage testimonials and forms.
  • Import testimonials manually by pasting the link to the source (e.g., a link to a Twitter post).
  • Create and fully customize forms which they can send to their clients to collect testimonials.
  • Embed forms and testimonials using a provided embed script for seamless integration.
  • Generate social media content, such as image posts and video compilations, using the testimonials collected.

For further details on the functionalities, refer to the documentation within the application.
